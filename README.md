# Multithreaded Task Scheduler (C++)

This project is an ongoing exploration of concurrency concepts through the implementation of a simple multithreaded task scheduler in C++.

The goal of the project is to understand how tasks can be executed efficiently across multiple threads while handling synchronization, load balancing, and shared state safely.

## Current Focus

- Implementing a basic **thread pool**
- Exploring **work-stealing queues** for load balancing
- Using **mutexes and condition variables** for synchronization
- Studying **deadlock scenarios and lock contention**
- Experimenting with **sharding strategies** to reduce shared dependencies

## Motivation

Modern systems rely heavily on parallel execution.  
This project is a hands-on attempt to better understand how scheduling, synchronization, and task distribution work inside multithreaded systems.

## Status

Work in progress — this repository is being updated as new experiments and improvements are implemented.
